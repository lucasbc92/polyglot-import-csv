Esta pasta conserva apenas os scripts de referência do fluxo Pandoc + ABNT.

O manuscrito e todos os artefatos (`abntex2.latex`, `references.bib`, CSL, imagens) estão no diretório pai **`..`** (`_docs/`).

Use `gerar-pdf.ps1` nesta pasta ou, de preferência, `.\_docs\scripts\gerar-pdf.ps1` a partir da raiz do repositório.
